## Overview

Reddit Listener is a Python-based tool that extracts real-time insights from Reddit and delivers them through a Slack bot interface. It uses AI to analyze Reddit content and provide:

- Top 10 questions people are asking
- Relevant keywords and phrases
- Community pain points with solutions
- AI-generated content ideas
